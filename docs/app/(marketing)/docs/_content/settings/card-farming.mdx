# Card Farming Settings

## Farm Games In Queue
Only farm cards for games you've added to your [Card Farming queue](/docs/features/card-farming)

SGI will ignore all other games in your library. Use this mode when you want precise control over which games get farmed.

<Callout type="info">
  **Farm Games In Queue** and **Farm All Games** are mutually exclusive — enabling one will disable the other.
</Callout>

## Farm All Games
Automatically farm cards for every owned game that still has drops remaining, ignoring the queue

SGI will go through all eligible games without you needing to queue them individually. Blacklisted games are still skipped even in this mode.

<Callout type="info">
  **Farm Games In Queue** and **Farm All Games** are mutually exclusive — enabling one will disable the other.
</Callout>

## Skip Unplayed Games
Skip games that have never been played *(0 playtime)* during card farming

<Callout type="info">
  **Skip Unplayed Games** and **Unplayed Games Only** are mutually exclusive — enabling one will disable the other.
</Callout>

## Unplayed Games Only
Only farm cards for games that have never been played *(0 playtime)*

<Callout type="info">
  **Skip Unplayed Games** and **Unplayed Games Only** are mutually exclusive — enabling one will disable the other.
</Callout>

## Drop Order
Choose which games to farm first when there's more than one queued

- **Highest Drops First** — prioritizes games with the most card drops remaining
- **Lowest Drops First** — prioritizes games with the fewest card drops remaining

## Automatically Farm Cards
Periodically check for owned games with card drops remaining and automatically start farming them *(Pro — Gamer tier)*

When enabled, SGI monitors your library on a recurring schedule and starts a farming cycle without you needing to trigger it manually.

## Start Another Task When Done
Automatically start another feature once Card Farming finishes

Enable the checkbox and choose either **Achievement Unlocker** or **Auto-Idle** from the dropdown to have it start automatically once your current farming cycle completes.
