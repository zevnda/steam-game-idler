# General Settings

You'll find these options under <MockButton type='context-cog' content='Settings → General' />.

## Always Online
Prevents Steam from setting your status as away while SGI is running *(Legacy Sign In only)*

When enabled, SGI will automatically switch your Steam status back to online if it changes to away, offline, or invisible while the app is running. This is useful when you want to appear active on Steam while games are idling in the background.

Only available for accounts using [Legacy Sign In](/docs/get-started/how-to-sign-in) (a real local Steam client). Accounts signed in with Steam don't need this. See [Online Status](#online-status) below, which covers the same need without any periodic re-checking required.

## Online Status *(Sign in with Steam only)*
Choose what online status friends see while signed in

Pick which status (Online, Away, Busy, Snooze, Looking to Trade, Looking to Play, Invisible, or Offline) your account shows to friends. Unlike Legacy Sign In's Always Online above, this status doesn't drift back to Away on its own, so there's nothing to periodically refresh. It's set once and stays that way until you change it.

## Custom Status While Idling *(Sign in with Steam only · Pro Gamer tier)*
Replace the "Playing [game]" text friends see while idling

Set your own custom message to show instead of the game's name while idling. You can set this before, during, or after idling starts. It only becomes visible to friends once you're actually idling a game you own.

## Show Games Only *(Sign in with Steam only)*
Hide DLC, soundtracks, videos, and tools from your library

When enabled, only owned games and family-shared games are shown in your library, filtering out non-game items like DLC, soundtracks, videos, and tools. Changing this refreshes your games list.

## Run At Startup
Automatically launch SGI when your computer starts

SGI will be registered to launch at login (Windows startup programs, or your Linux desktop's autostart mechanism) so it runs in the background whenever you log in, without needing to open it manually. Works best when paired with [Start Minimized](#start-minimized).

## Start Minimized
Launch SGI to the system tray instead of showing the main window

When enabled, SGI opens silently in the background on startup. You can access it at any time by clicking the SGI icon in the system tray. Pairs well with [Run At Startup](#run-at-startup) for a fully hands-off setup.

## Close to Tray
Keep SGI running when you close the window

When enabled, clicking the X button minimizes SGI to the system tray instead of exiting it. To fully close the app, right-click the system tray icon and select `Quit`.

## Automatically Update Games List
Silently refresh your games list in the background so it stays current without a manual refresh *(Pro Casual tier)*

When enabled, SGI periodically checks for changes to your Steam library, such as newly purchased or removed games, and updates the list without requiring a manual refresh. The check runs every 5 minutes by default, or every minute if you've set your own [Steam Web API Key](#steam-web-api-key) below (the shared built-in key checks less often to stay within Steam's rate limits).

## Language
Choose the display language for SGI

The language is set automatically based on your system settings, but you can change it to any supported language from the dropdown. If you'd like to help translate SGI into your language, see the [contribution guide](https://github.com/zevnda/steam-game-idler/discussions/148).

## Steam Web API Key
Use your own Steam Web API key to access private profile data

By default, SGI fetches your profile and game data from public Steam APIs. Providing your own API key lets SGI access data even when your Steam profile is set to private.

- API keys are free for all Steam accounts. [Get yours here](https://steamcommunity.com/dev/apikey)
- The API key must belong to the same Steam account you're signed into in SGI. If you use multiple accounts, you'll need to set the key for each one separately
- Some features may still be limited with a private profile even with an API key
- Your key is validated against Steam before it's saved, and stored securely rather than in a plain settings file
