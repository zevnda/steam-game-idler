# Customization Settings

## Disable Tooltips
Hide the small informational pop-ups that appear when hovering over elements in the app

Tooltips provide extra context for buttons and settings, but can feel repetitive once you're familiar with the app. Enable this to turn them off.

## Show Recommended Games
Show or hide the recommended games carousel on the **Your Games** page

The recommended carousel surfaces games in your library that you haven't played yet. Disable this if you prefer a cleaner view.

## Show Recently Played Games
Show or hide the recently played games carousel on the **Your Games** page

The recently played carousel shows the last few games you've run through Steam. Disable this if you don't need quick access to recent games.

## Font
Change the font used throughout the SGI interface *(non-default fonts require Pro — Casual tier)*

Every font is bundled with the app, so switching never needs a download. Pro fonts are unlocked with a [Casual or Gamer subscription](/pro).

## Background Image
Set a custom background image for the SGI window *(Pro — Casual tier)*

You can upload any common image file (PNG, JPG, WEBP, etc.) from your computer to use as a background. The image is applied as a subtle overlay behind the main UI. Click **Clear** to remove the background and return to the default appearance.
