# Keybinds

You'll find this reference list under <MockButton type='context-cog' content='Settings → Keybinds' />. It's read-only. Shortcuts can't be remapped.

## Zoom

| Action | Shortcut |
|---|---|
| Zoom in | `Ctrl` `+` (or `Ctrl` + scroll up) |
| Zoom out | `Ctrl` `-` (or `Ctrl` + scroll down) |
| Reset zoom | `Ctrl` `0` |

## Navigation

| Action | Shortcut |
|---|---|
| Next page | `Ctrl` `]` |
| Previous page | `Ctrl` `[` |
| Open search | `/` |
| Toggle settings | `Ctrl` `,` |
| Toggle sidebar | `Ctrl` `W` |
