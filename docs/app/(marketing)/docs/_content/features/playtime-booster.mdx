# How to Idle Steam Games and Boost Playtime

Idling a game increases its playtime hours without you needing to actually play it. This is useful for reaching the playtime needed to start receiving trading card drops, padding your Steam profile stats, or simply keeping a game "running" in the background.

<Callout type="info">
  You can idle a maximum of 32 games simultaneously, across every SGI feature combined. This limit is set by the Steam client.
</Callout>

## Start Idling a Game

<Steps>
  <Step>
    Click <MockButton type='your-games' /> in the sidebar
  </Step>

  <Step>
    Click the <MockButton type='start-idle' /> button on individual games to begin idling them
  </Step>

  <Step>
    Click <MockButton type='idling-games' /> in the sidebar to see everything currently idling
  </Step>
</Steps>
