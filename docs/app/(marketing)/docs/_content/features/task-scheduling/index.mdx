# How to Schedule Automated Steam Game Tasks

Task Scheduling lets you chain SGI's automation features together so one starts automatically when the previous one finishes. For example, you can have Card Farming hand off to Achievement Unlocker, or Achievement Unlocker hand off to Automatic Idler — all without any manual intervention.

<DocsCTA />

<Callout type="info">
  Tasks run one at a time, in sequence. When the first task finishes, the next one begins automatically.
</Callout>

## Setting Up Task Scheduling

<Steps>
  <Step>
    Click <MockButton type='context-cog' content='Settings' /> in the sidebar
  </Step>

  <Step>
    Go to the **Card Farming** tab and enable the `Schedule Next Task` option, then select which feature should start when Card Farming completes
  </Step>

  <Step>
    Go to the **Achievement Unlocker** tab and enable the `Schedule Next Task` option, then select which feature should start when Achievement Unlocker completes
  </Step>
</Steps>

Each feature has its own `Schedule Next Task` setting — configure whichever ones you want in the chain. You don't need to configure both unless you're chaining all three features together.

See the individual settings pages for details:
- [Card Farming Settings → Schedule Next Task](/docs/settings/card-farming#schedule-next-task)
- [Achievement Unlocker Settings → Schedule Next Task](/docs/settings/achievement-unlocker#schedule-next-task)
