# How to Idle Steam Games and Boost Playtime

Idling a game increases its playtime hours without you needing to actually play it. This is useful for reaching the playtime needed to start receiving trading card drops, padding your Steam profile stats, or simply keeping a game "running" in the background.

<DocsCTA />

<Callout type="info">
  You can idle a maximum of 32 games simultaneously, across every SGI feature combined (manual idling, [Auto-Idle](/docs/features/auto-idle), [Card Farming](/docs/features/card-farming), and [Achievement Unlocker](/docs/features/achievement-unlocker)). This limit is set by the Steam client.
</Callout>

## Start Idling a Game

<Steps>
  <Step>
    Click <MockButton type='your-games' /> in the sidebar
  </Step>

  <Step>
    Click <MockButton type='start-idle' /> below a game's image to begin idling it. You can do this for multiple games
  </Step>

  <Step>
    Click <MockButton type='idling-games' /> in the sidebar to see everything currently idling, along with how long each has been running
  </Step>
</Steps>

## The Idling Page

Every game currently idling (whether started manually, or by Auto-Idle, Card Farming, or the Achievement Unlocker) shows up on the <MockButton type='idling-games' /> page, grouped into a section per feature. If a game is claimed by more than one feature at once (e.g. manually started while also queued for Auto-Idle), it's only shown once, under whichever feature has priority.

## Stopping Idling

To stop a single game, click its <MockButton type='stop' /> button on either <MockButton type='your-games' /> or <MockButton type='idling-games' />.

To stop everything one feature is idling, click that section's <MockButton type='stop' /> button on the <MockButton type='idling-games' /> page. To stop every idling game across every feature at once, click <MockButton type='stop-all' /> in the page header.

<Callout type="info">
  Idling only increases playtime hours. It does not automatically unlock achievements or farm trading cards. For those, see [Achievement Unlocker](/docs/features/achievement-unlocker) and [Card Farming](/docs/features/card-farming)
</Callout>

## Additional Information

When you idle a game, SGI opens a background helper process for each game to register it as active with Steam (or, when [signed in with Steam](/docs/get-started/how-to-sign-in), announces it via one shared session instead of a process per game). You don't need to manage these yourself. They clean up automatically when you stop idling or exit SGI.
