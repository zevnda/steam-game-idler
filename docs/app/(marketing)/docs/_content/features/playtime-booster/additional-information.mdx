# Additional Information

When you idle a game, SGI opens a background `SteamUtility.exe (idle)` process for each game to register as active with Steam. **Do not close these windows manually** — they will close on their own when you stop idling or exit SGI.

Playtime Booster only increases hours — it does not automatically unlock achievements or farm trading cards. For those, see [Achievement Unlocker](/docs/features/achievement-unlocker) and [Card Farming](/docs/features/card-farming).
