# How to Unlock Steam Achievements Automatically

The Achievement Unlocker automatically unlocks achievements for your Steam games over time, with customizable delays and ordering to mimic natural gameplay.

<DocsCTA />

<Callout type="info">
    Configure this feature's settings in [Settings → Achievement Unlocker](/docs/settings/achievement-unlocker) before using it
</Callout>

<Callout type="info">
    PRO subscribers can unlock achievements for multiple games at the same time by enabling [Run Multiple Games](/docs/settings/achievement-unlocker#run-multiple-games) — up to 3 at once on Casual, or up to 32 on Gamer
</Callout>

### Default Mode

In default mode, achievements are unlocked in the order of global achievement percentage *(only available when [signed in via Legacy Sign In](/docs/get-started/how-to-sign-in#whats-different-between-the-two))*, with the delay set in [Settings → Achievement Unlocker → Unlock Interval](/docs/settings/achievement-unlocker#unlock-interval).

<Steps>
  <Step>
    Click <MockButton type='achievement-unlocker' /> in the sidebar
  </Step>

  <Step>
    Go to the <MockButton type='all-games' /> tab and click on individual games to add and remove them
  </Step>

  <Step>
    Optionally, you can customize the unlock order and set delays for individual achievements by [following these steps](/docs/features/achievement-unlocker/custom-order-and-unlock-delay)
  </Step>

  <Step>
    Click <MockButton type='achievement-unlocker-action' />
  </Step>
</Steps>

While running, each active game shows a live countdown to its next achievement, with the next several upcoming achievements listed below it.
