# How to Unlock Steam Achievements Automatically

The Achievement Unlocker automatically unlocks achievements for your Steam games over time, with customizable delays and ordering to mimic natural gameplay.

<Callout type="info">
    Configure this feature's settings in [Settings → Achievement Unlocker](/docs/settings/achievement-unlocker) before using it
</Callout>

### Default Mode

In default mode, achievements are unlocked in order of global achievement rarity (most common first), with the delay set in [<MockButton type='context-cog' content='Settings → Achievement Unlocker → Unlock Interval' />](/docs/settings/achievement-unlocker#unlock-interval).

If you want to define a custom order, and dial in more accurate delays, use the [Custom Order & Unlock Delay](/docs/features/achievement-unlocker/custom-order-and-unlock-delay)

<Steps>
  <Step>
    Click <MockButton type='achievement-unlocker' /> in the sidebar
  </Step>

  <Step>
    Go to the <MockButton type='all-games' /> tab and click the <MockButton type='context-add' /> button on individual games to add them to your queue
  </Step>

  <Step>
    Optionally, you can customize the unlock order and set delays for individual achievements by [following these steps](/docs/features/achievement-unlocker/custom-order-and-unlock-delay)
  </Step>

  <Step>
    Switch to the <MockButton type='content' content='Queue' /> tab to review what you've added, then click <MockButton type='achievement-unlocker-action' />
  </Step>
</Steps>

While running, each active game shows a live countdown to its next achievement, with the next several upcoming achievements listed below it.
