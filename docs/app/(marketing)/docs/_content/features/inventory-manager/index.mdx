# How to Manage and Sell Steam Inventory Items Automatically

The Inventory Manager feature allows you to manage your Steam inventory directly within SGI, including the ability to view all your marketable items (trading cards, foil cards, emoticons, backgrounds, and more) and sell them on the Steam marketplace.

<DocsCTA />

<Callout type="warn">
    This feature uses an undocumented Steam API endpoint with strict rate limits to fetch card prices. If you encounter errors while selling cards or fetching card prices, please try increasing the [Sell Delay](/docs/settings/inventory-manager#sell-delay), or wait a few hours before trying  again
</Callout>

<Callout type="info">
    You must add your [Steam Credentials](/docs/steam-credentials) before you can use this feature, unless you [signed in with Steam](/docs/get-started/how-to-sign-in) on a **Gamer** tier account, in which case this happens automatically
</Callout>

<Callout type="info">
    Configure this feature's settings in [Settings → Inventory Manager](/docs/settings/inventory-manager) before using it
</Callout>

<Steps>
  <Step>
    Click <MockButton type='inventory-manager' /> in the sidebar
  </Step>

    #### List individual items
    <Steps>
      <Step>
        Find the item you want to sell and input a sale price
      </Step>
      <Step>
        Click <MockButton type='list-card' /> to list the item
      </Step>
    </Steps>

    #### List selected items
    <Steps>
      <Step>
        Find the items you want to sell and input a sale price for each one
      </Step>
      <Step>
        Click <MockButton type='list-selected' /> to list only the selected items
      </Step>
    </Steps>

    #### List all items
    <Steps>
      <Step>
        Click <MockButton type='list-all' /> to list all items
      </Step>
    </Steps>

    #### List duplicate items *(Pro Gamer tier)*
    <Steps>
      <Step>
        Click <MockButton type='sell-dupes' /> to automatically list all duplicate items for sale, keeping one copy of each
      </Step>
    </Steps>

  <Step>
    Confirm each listing via the Steam mobile app or email depending on your [Steam Guard](https://help.steampowered.com/en/faqs/view/06B0-26E6-2CF8-254C) settings
  </Step>

  <Step>
    Click <MockButton type='content' content='Refresh' /> in the page header to update your card list
  </Step>
</Steps>

Foil (holographic) items are shown with a shimmer effect on their card, and every item links out to [SteamCardExchange](https://www.steamcardexchange.net/) for reference pricing.
