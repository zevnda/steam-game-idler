# Build It Yourself

SGI's source code is fully public on [GitHub](https://github.com/zevnda/steam-game-idler). The desktop app is a [Tauri](https://tauri.app/) app (Rust + Next.js), and it drives a bundled helper process, `SteamUtility`, built from a separate C#/.NET project at `libs/SteamUtility/`.

## Prerequisites

* [Node.js](https://nodejs.org/en/download/prebuilt-installer) >= 20.0
* [pnpm](https://pnpm.io/installation) - the project uses pnpm workspaces, not npm/yarn
* [Rust](https://www.rust-lang.org/tools/install) (stable toolchain)
* [.NET SDK](https://dotnet.microsoft.com/en-us/download) >= 8.0 (`libs/SteamUtility` targets `net8.0` only)

**Windows only:**
* [VS Build Tools](https://visualstudio.microsoft.com/visual-cpp-build-tools/) = latest, with the "Desktop development with C++" workload
* [WebView2](https://developer.microsoft.com/en-us/Microsoft-edge/webview2/) = latest (*pre-installed on Windows 11 and newer Windows 10 versions*)

**Linux only:** 
* [Tauri's Linux prerequisite](https://v2.tauri.app/start/prerequisites/#linux) system dependencies for your distro

<Steps>
  <Step>
    ### Clone Repo
    ```bash
    git clone https://github.com/zevnda/steam-game-idler.git
    cd ./steam-game-idler
    ```
  </Step>

  <Step>
    ### Install Dependencies
    ```bash
    pnpm install
    ```

    This installs dependencies for both the root app and the `docs/` workspace package
  </Step>

  <Step>
    ### Build SteamUtility

    ```bash
    dotnet build ./libs/SteamUtility/SteamUtility.csproj
    ```

    This outputs `SteamUtility` to `src-tauri/target/debug/libs/` (or `release/libs/` for a release build); `pnpm tauri dev`/`pnpm tauri build` expect it to already be there
  </Step>

  <Step>
    ### Create Environment Files

    Create a `.env` file in the root of `./steam-game-idler` and add a `KEY` variable containing your [Steam Web API Key](https://steamcommunity.com/dev/apikey)

    <Files>
      <Folder name="steam-game-idler" defaultOpen>
        <Folder name="public" />
        <Folder name="libs" />
        <Folder name="docs" />
        <Folder name="src" defaultOpen>
            <Folder name="pages" />
            <Folder name="features" />
            <Folder name="shared" />
        </Folder>
        <Folder name="src-tauri" defaultOpen>
            <Folder name="tauri.conf.json" />
        </Folder>
        <File name=".env" className='bg-fd-primary/10 text-fd-primary hover:bg-fd-primary/20 hover:text-fd-primary' />
        <File name="pnpm-lock.yaml" />
        <File name="package.json" />
      </Folder>
    </Files>

    ```ini title=".env" copy
    KEY="ABC123******789XYZ"
    ```
  </Step>

  <Step>
    ### Development or Production Build

    ```bash
    # Development
    pnpm td       # shorthand for `tauri dev`

    # Production
    pnpm tb       # shorthand for `tauri build`, outputs to `src-tauri/target/release/bundle/`
    ```
  </Step>
</Steps>

## Other Useful Scripts

Run these from the repo root:

| Script | What it does |
|---|---|
| `pnpm typecheck` | `tsc --noEmit` for the desktop app |
| `pnpm lint` | ESLint for the desktop app |
| `pnpm dd` | Starts the docs site dev server (this site) |
| `pnpm build:docs` | Production build of the docs site |
| `pnpm build:libs` | Builds the `./libs/SteamUtility.csproj` |
| `pnpm i18n:check` | Checks `en-US.json` for duplicate translation values |
