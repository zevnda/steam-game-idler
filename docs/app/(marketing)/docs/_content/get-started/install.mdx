# Install

SGI runs on **Windows** (10/11, 64-bit) and **Linux** (64-bit, via `.deb`, `.rpm`, or AppImage). On Windows, SGI can be used via either the [installer](/docs/get-started/install#installer-version) or [portable](/docs/get-started/install#portable-version) version — you can use the portable version from a USB drive or any folder.

<DocsCTA />

<Callout type='info'>
    Linux support is agent-mode sign-in only ([Sign in with Steam](/docs/get-started/how-to-sign-in#sign-in-with-steam)) - Legacy Sign In, which needs a real local Steam client, is Windows-only
</Callout>

## Installer Version

<Steps>
  <Step>
    ### Download
    Download `Steam.Game.Idler_X.X.X_x64-setup.exe` from the [latest release](https://github.com/zevnda/steam-game-idler/releases/latest/) page on GitHub
  </Step>

  <Step>
    ### Install
    Run the installer and follow the steps of the installation guide
  </Step>

  <Step>
    ### Run
    After installation, you can launch SGI in one of three ways:
      * Select "Run Steam Game Idler" and click "Finish" when prompted
      * Run `Steam Game Idler.exe` from the installation directory
        * Defaults to `C:/Program Files/Steam Game Idler`
      * Use the desktop shortcut created by the installer
  </Step>
</Steps>

## Portable Version

<Callout type='warn'>
    The portable version does not support automatic updates. You'll need to download and replace the executable manually for each new release
</Callout>

<Steps>
  <Step>
    ### Download
    Download `Steam.Game.Idler_X.X.X_x64-portable.zip` from the [latest release](https://github.com/zevnda/steam-game-idler/releases/latest/) page on GitHub
  </Step>

  <Step>
    ### Extract
    Extract the contents of the `portable.zip` file to any folder you want
  </Step>

  <Step>
    ### Run
    Open the folder and run `Steam Game Idler.exe`
  </Step>
</Steps>

## Linux

`.deb` and `.rpm` are the **recommended** Linux install method for performance — the AppImage works on any distro with no install step, but runs under a compatibility layer (XWayland) that's noticeably slower than a native `.deb`/`.rpm` install on a Wayland desktop. The trade-off: only the AppImage supports in-app automatic updates (the same way Windows does) — a `.deb`/`.rpm` install is managed by your system's package manager instead, so SGI can't silently update itself; download and install the new package manually when a new release comes out.

<Steps>
  <Step>
    ### Download
    Grab the file matching your distro/format from the [latest release](https://github.com/zevnda/steam-game-idler/releases/latest/) page on GitHub: `steam-game-idler_X.X.X_amd64.deb` (Debian/Ubuntu-based), `steam-game-idler-X.X.X.x86_64.rpm` (Fedora/openSUSE-based), or the `.AppImage` (any distro)
  </Step>

  <Step>
    ### Install
    * **`.deb`**: `sudo dpkg -i steam-game-idler_X.X.X_amd64.deb`
    * **`.rpm`**: `sudo rpm -i steam-game-idler-X.X.X.x86_64.rpm`
    * **AppImage**: make it executable (`chmod +x Steam.Game.Idler_X.X.X_amd64.AppImage`) and run it directly — no installation needed
  </Step>

  <Step>
    ### Run
    Launch SGI from your desktop's application launcher (`.deb`/`.rpm`), or by running the AppImage file directly
  </Step>
</Steps>

<Callout type='warn'>
    A `.deb`/`.rpm` install won't notify you of new releases. Check the [latest release](https://github.com/zevnda/steam-game-idler/releases/latest/) page periodically, or use the AppImage instead if you'd rather have automatic updates
</Callout>
