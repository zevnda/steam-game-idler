# Install

SGI supports **Windows** (10/11, 64-bit) and **Linux** (64-bit). Follow the steps below for your OS and install method.

<DocsCTA />

## Windows Installer

<Steps>
  <Step>
    ### Download
    Download `Steam.Game.Idler_X.X.X_x64-setup.exe` from the [latest release](https://github.com/zevnda/steam-game-idler/releases/latest/) page on GitHub
  </Step>

  <Step>
    ### Install
    Run the installer and follow the steps of the installation guide
  </Step>

  <Step>
    ### Run
    After installation, you can launch SGI in one of three ways:
      * Select "Run Steam Game Idler" and click "Finish" when prompted
      * Run `Steam Game Idler.exe` from the installation directory
        * Defaults to `C:/Program Files/Steam Game Idler`
      * Use the desktop shortcut created by the installer
  </Step>
</Steps>

## Windows Portable

<Callout type='warn'>
    The portable version does not support automatic updates. You'll need to download and replace the executable manually for each new release
</Callout>

<Steps>
  <Step>
    ### Download
    Download `Steam.Game.Idler_X.X.X_x64-portable.zip` from the [latest release](https://github.com/zevnda/steam-game-idler/releases/latest/) page on GitHub
  </Step>

  <Step>
    ### Extract
    Extract the contents of the `portable.zip` file to any folder you want
  </Step>

  <Step>
    ### Run
    Open the folder and run `Steam Game Idler.exe`
  </Step>
</Steps>

## Linux

<Callout type='info'>
    Linux supports [Sign in with Steam](/docs/get-started/how-to-sign-in#sign-in-with-steam) only
</Callout>

<Callout type='warn'>
    The `.deb` / `.rpm` versions do not support automatic updates. You'll need to download and install the new package manually for each release
</Callout>

<Steps>
  <Step>
    ### Download
    Grab the file matching your distro/format from the [latest release](https://github.com/zevnda/steam-game-idler/releases/latest/) page on GitHub:
    - **Debian/Ubuntu-based**: `Steam.Game.Idler_X.X.X_amd64.deb`
    - **Fedora/openSUSE-based**: `Steam.Game.Idler_X.X.X.x86_64.rpm`
    - **Any distro**: `Steam.Game.Idler_X.X.X_amd64.AppImage`
  </Step>

  <Step>
    ### Install
    * **`.deb`**: `sudo dpkg -i Steam.Game.Idler_X.X.X_amd64.deb`
    * **`.rpm`**: `sudo rpm -i Steam.Game.Idler_X.X.X.x86_64.rpm`
    * **AppImage**: make it executable (`chmod +x Steam.Game.Idler_X.X.X_amd64.AppImage`) and run it directly — no installation needed
  </Step>

  <Step>
    ### Run
    Launch SGI from your desktop's application launcher (`.deb`/`.rpm`), or by running the AppImage file directly
  </Step>
</Steps>
