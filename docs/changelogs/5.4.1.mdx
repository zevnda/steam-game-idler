### Improved 🚀
- Moved sorting and filtering in **Inventory Manager** out of the page header and into a dedicated panel, making it easier to browse and combine filters
- **Achievement Unlocker** now shows scan progress while checking a large queue of games for achievements, instead of briefly flashing each game as "currently farming" before discarding it

### Fixed 🐛
- Fixed an issue in **Achievement Unlocker** where games could fail to be removed from the queue when many games were processed at the same time
