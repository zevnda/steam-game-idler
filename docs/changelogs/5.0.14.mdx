### New 🎉
- **New PRO Gamer Tier Feature**: Added [**Import Timings**](https://steamgameidler.com/docs/features/achievement-unlocker#import-timings) to the **Custom Order & Unlock Delay** page in **Achievement Unlocker**
  - This feature lets you copy the unlock order and achievement delays from another Steam user's profile. This is useful for matching a specific playstyle or replicating naturally-timed unlocks
  - Enter a Steam profile (SteamID64, profile URL, or username) to fetch their achievement unlock history
  - Achievements are automatically reordered to match the sequence they were unlocked in by the target profile
  - Per-achievement delays are calculated from the exact time between each consecutive unlock
  - Any achievement not unlocked by the target profile will be skipped
  - **The target profile and games list must be set to public on Steam**. You can confirm this by visiting their games list in an incognito browser tab
