### New 🎉
- Added an in-app banner system to display important messages to users. This will be useful for announcing important changes, issues affecting SGI, new features, and promotions
- Added support for the **Slovenian** language. If you would like to contribute a new translation or improve an existing one, please refer to the **[translation guide](https://github.com/zevnda/steam-game-idler/discussions/148)** for information on how to get started
