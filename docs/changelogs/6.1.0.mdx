## Linux Support Is Here!

Steam Game Idler now runs natively on **Linux**. It's been one of the more highly requested features since the app's early days, and two years on, it's finally here. Thanks for your patience.

### New 🎉
- GitHub release assets now contain an `.AppImage`, `.deb`, or `.rpm`, whichever suits your distro. Grab one from the [releases page](https://github.com/zevnda/steam-game-idler/releases/latest) and see the [Linux install guide](https://steamgameidler.com/docs/get-started/install#linux) for setup steps
- The `.AppImage` supports automatic updates, the same as the Windows installer. The `.deb` and `.rpm` packages don't, since they're managed by your system's package manager, you'll need to download and install each new release manually
- Added support for the **French** language. If you'd like to help translate SGI into your language, see the [contribution guide](https://github.com/zevnda/steam-game-idler/discussions/148)

### Good to know
- Linux only supports the **Steam Sign-in** method
- A small number of Game Coordinator titles (TF2, Dota 2, CS2, Left 4 Dead 2, Portal 2) aren't supported on Linux
- A handful of small, accepted Linux-only quirks (no rounded window corners, an extra click sometimes needed after dragging the window, and a couple of others) are documented in the [Linux quirks & limitations section](https://github.com/zevnda/steam-game-idler/wiki#known-linux-quirks--limitations) if you run into anything that looks off
