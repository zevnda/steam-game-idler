### Improved 🚀
- The inputs in **Custom Order & Unlock Delay** now support 2-decimal place numbers (e.g. `0.01`) to allow for better "to the exact second" precision when unlocking achievements

### Fixed 🔧
- **Inventory Manager's** **Sell all** and **Sell dupes** now use a card's custom price when one has been set instead of always falling back to the highest buy/lowest sell order price
