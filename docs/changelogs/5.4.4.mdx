### New 🎉
- Added a **Global Max Card Farming Time** setting to **Settings > Game Settings**, allowing users to set a default maximum farming duration that applies to all games in the **Card Farming** feature
- Added a sort dropdown to the **Drops Remaining** tab in **Card Farming**

### Improved 🚀
- The selected sort option in the **All Games** tab of custom lists is now remembered when switching pages, instead of resetting to default

### Fixed 🐛
- Fixed an issue in **Inventory Manager** where locking an item that had duplicates would also lock all of its duplicates instead of just the single item
- Fixed an issue where the **Automated Method** for Steam credentials couldn't be used with more than one Steam account, as signing in or revalidating always used the session from whichever account was last signed in via the sign-in window
