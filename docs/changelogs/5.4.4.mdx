### New 🎉
- Added a **Global Max Card Farming Time** setting to **Settings > Game Settings**, allowing users to set a default maximum farming duration that applies to all games in the **Card Farming** feature
- Added a sort dropdown to the **Drops Remaining** tab in **Card Farming**

### Improved 🚀
- The selected sort option in the **All Games** tab of custom lists is now remembered when switching pages, instead of resetting to default
