# How to Manage Steam Achievements Manually

The Steam Achievement Manager lets you manually unlock and lock achievements for any game in your library, as well as modify game statistics. Unlike the [Achievement Unlocker](/docs/features/achievement-unlocker), this is a one-by-one manual tool — useful for targeting specific achievements without automation.

<DocsCTA />

### Achievements

<Steps>
  <Step>
    Click <MockButton type='your-games' /> in the sidebar
  </Step>

  <Step>
    Click <MockButton type='achievement-manager' /> beneath the game image to open the achievement manager
  </Step>

  <Step>
    On the achievement manager screen you will see a list of achievements for that game, if any are available

    Click <MockButton type='unlock' /> or <MockButton type='lock' /> next to an achievement to unlock or lock that specific achievement

    Click <MockButton type='unlock-all' /> or <MockButton type='lock-all' /> in the page header to unlock or lock all achievements for that game at once
  </Step>
</Steps>

### Statistics

<Steps>
  <Step>
    Click <MockButton type='your-games' /> in the sidebar
  </Step>

  <Step>
    Click <MockButton type='achievement-manager' /> beneath the game image to open the achievement manager
  </Step>

  <Step>
    On the achievement manager screen, click the <MockButton type='content' content='Statistics' /> tab. You will see a list of statistics for that game, if any are available
  </Step>

  <Step>
    Change the `value` of any statistics you want to modify and click <MockButton type='save-changes' />
  </Step>
</Steps>

## Special Flags

Some achievements and statistics have restrictions set by the game developer that affect what you can change. These are indicated directly in the UI.

- **Protected:** Only the game's official servers can modify these. The unlock/lock buttons will be disabled for protected achievements
- **IncrementOnly:** This statistic can only increase in value — you can raise it from 5 to 6, but not lower it from 5 to 4
- **Hidden:** The achievement's description is blurred by default — hover over it to reveal the text

For statistics, any active flags are shown directly beneath the stat value in the Statistics tab.
