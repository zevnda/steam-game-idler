# How to Boost Steam Game Playtime

The Playtime Booster lets you idle one or more Steam games simultaneously to increase their playtime hours. This is useful for reaching the playtime needed to start receiving trading card drops, padding your Steam profile stats, or simply keeping a game "running" in the background.

<DocsCTA />

<Callout type="info">
  You can idle a maximum of 32 games simultaneously. This limit is set by the Steam client.
</Callout>

## Start Idling

<Steps>
  <Step>
    Click <MockButton type='your-games' /> in the sidebar
  </Step>

  <Step>
    Click <MockButton type='start-idle' /> below a game's image to begin idling it. You can do this for multiple games
  </Step>

  <Step>
    Click <MockButton type='idling-games' /> in the sidebar to see all games currently being idled, along with how long each has been running
  </Step>
</Steps>
