# Possible Task Chains

| When this finishes... | ...this can start automatically |
|---|---|
| Card Farming | Achievement Unlocker |
| Card Farming | Automatic Idler |
| Achievement Unlocker | Card Farming |
| Achievement Unlocker | Automatic Idler |

You can chain further by configuring both handoffs. For example: Card Farming → Achievement Unlocker → Automatic Idler works if you set Card Farming to hand off to Achievement Unlocker, and Achievement Unlocker to hand off to Automatic Idler.
