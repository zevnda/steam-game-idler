# Pricing Details

When you use <MockButton type='list-all' />, SGI will list each item at its `highest_buy_order`. If that price isn't available, it will use its `lowest_sell_order`. If neither price is available, the item won't be listed.

You can choose which option to use by default by changing the [Default Sell Option](/docs/settings/inventory-manager#default-sell-option) in the settings.
