# How to Unlock Steam Achievements Automatically

The Steam Achievement Unlocker feature allows you to automatically unlock achievements for your Steam games over time, with customizable delays and ordering to mimic natural gameplay.

<DocsCTA />

<Callout type="info">
    Configure this feature's settings in [Settings → Achievement Unlocker](/docs/settings/achievement-unlocker) before using it
</Callout>

### Default Mode

In default mode, achievements are unlocked in the order of global achievement percentage, with the delay set in [Settings → General → Unlock Interval](/docs/settings/achievement-unlocker#unlock-interval).

<Steps>
  <Step>
    Click <MockButton type='achievement-unlocker' /> in the sidebar
  </Step>

  <Step>
    Go to the <MockButton type='all-games' /> tab and click on individual games to add and remove them
  </Step>

  <Step>
    Optionally, you can customize the unlock order and set delays for individual achievements by [following these steps](#custom-order--unlock-delay)
  </Step>

  <Step>
    Click <MockButton type='achievement-unlocker-action' />
  </Step>
</Steps>

### Custom Order & Unlock Delay

The `custom order` feature lets you set the exact order in which achievements are unlocked, and allows you to skip any that you don’t want unlocked.

The `unlock delay` is how long SGI waits after unlocking one achievement before unlocking the next.

> If you set a delay of 10 minutes on achievement one, SGI will unlock achievement one, wait 10 minutes, and then unlock achievement two.

Unlock delays override the global value set in [Settings → General → Unlock Interval](/docs/settings/achievement-unlocker#unlock-interval). If no unlock delay is set, the global value is used.

You can customize the order, set delays, or both — each step is optional.

<Steps>
  <Step>
    Follow steps 1 and 2 from the [Default Mode](#default-mode) section above
  </Step>

  <Step>
    Click <MockButton type='unlock-order' /> below the game you want to customize
  </Step>

  <Step>
    Drag and drop achievements by grabbing the <MockButton type='grabber' /> to set their unlock order
  </Step>

  <Step>
    Click <MockButton type='checkbox' /> to uncheck achievements that you want to exclude from being unlocked
  </Step>

  <Step>
    The first row in the list is **Initial Delay** — enter a value (in minutes) to set how long SGI waits before unlocking the very first achievement
  </Step>

  <Step>
    Enter a delay (in minutes) in the input field of each achievement to set how long SGI waits after unlocking that achievement before moving to the next one
  </Step>

  <Step>
    Click <MockButton type='save-alt' /> and then click <MockButton type='achievement-unlocker-action' />
  </Step>
</Steps>

### Import Timings

The `import timings` feature lets you copy the unlock order and achievement delays from another Steam user's profile. This is useful for matching a specific playstyle or replicating naturally-timed unlocks.

<Callout type="info">
  The target profile and their game details must be set to public on Steam. You can verify this by viewing their games list in an incognito browser window.
</Callout>

<Steps>
  <Step>
    Follow steps 1 and 2 from the [Default Mode](#default-mode) section above
  </Step>

  <Step>
    Click <MockButton type='unlock-order' /> below the game you want to customize
  </Step>

  <Step>
    Click <MockButton type='import-timings' /> in the top-right of the custom order page
  </Step>

  <Step>
    Enter the target user's **SteamID64**, **profile URL**, or **username** and click <MockButton type='content' content='Import' />
  </Step>

  <Step>
    The achievement list will be reordered to match their unlock sequence, with delays calculated from the time between each unlock. Any achievement not unlocked by the target profile will be skipped. You can still adjust the order and delays manually before saving
  </Step>

  <Step>
    Click <MockButton type='save-alt' /> and then click <MockButton type='achievement-unlocker-action' />
  </Step>
</Steps>
