# Build It Yourself

## Prerequisites

* [Node.js](https://nodejs.org/en/download/prebuilt-installer) >= 18.0
* [Rustup](https://www.rust-lang.org/tools/install) = latest
* [.NET SDK](https://dotnet.microsoft.com/en-us/download) >= 8.0
* [.NET Framework Developer Pack](https://dotnet.microsoft.com/en-us/download/dotnet-framework/net48) = 4.8
* [VS Build Tools](https://visualstudio.microsoft.com/visual-cpp-build-tools/) = latest
* [WebView2](https://developer.microsoft.com/en-us/Microsoft-edge/webview2/) = latest - *pre-installed on Windows 11 and the latest version of Windows 10*

<Steps>
  <Step>
    ### Clone Repo And Submodules
    ```npm
    git clone --recursive https://github.com/zevnda/steam-game-idler.git
    ```
  </Step>

  <Step>
    ### Install Dependencies

    ```npm
    cd ./steam-game-idler
    npm run install
    ```
  </Step>

  <Step>
    ### Build Libraries

    ```npm
    dotnet build ./libs/SteamUtility.csproj # Outputs to `/src-tauri/libs/`
    ```
  </Step>

  <Step>
    ### Create Environment Files

    Create a `.env.development` and `.env.production` file in the root of `steam-game-idler`

    Add a `KEY` variable containing your [Steam Web API Key](https://steamcommunity.com/dev/apikey) to both `.env.development` and `.env.production`

    <Files>
      <Folder name="steam-game-idler" defaultOpen>
        <Folder name="public" />
        <Folder name="libs" />
        <Folder name="src" defaultOpen>
            <Folder name="pages" />
            <Folder name="components" />
        </Folder>
        <Folder name="src-tauri" defaultOpen>
            <Folder name="tauri.conf.json" />
        </Folder>
        <File name=".env.development" className='bg-fd-primary/10 text-fd-primary hover:bg-fd-primary/20 hover:text-fd-primary' />
        <File name=".env.production" className='bg-fd-primary/10 text-fd-primary hover:bg-fd-primary/20 hover:text-fd-primary' />
        <File name="pnpm-lock.yaml" />
        <File name="package.json" />
      </Folder>
    </Files>

    ```ini title=".env.development" copy
    KEY="ABC123******789XYZ"
    ```

    ```ini title=".env.production" copy
    KEY="ABC123******789XYZ"
    ```
  </Step>

  <Step>
    ### Start in Dev Mode or Build App

    ```npm
    # Start in dev mode, or
    npm run tauri dev

    # Build the bundle
    npm run tauri build # Outputs to `/src-tauri/target/release/bundle/`
    ```
  </Step>
</Steps>
