# Achievement Unlocker Settings

## Idle Game
Keep the game running in the background while achievements are being unlocked

When enabled, SGI will idle the current game during the unlocking process, increasing its playtime alongside the achievements. This makes the activity look more consistent with natural gameplay.

## Skip Hidden Achievements
Skip achievements that the game developer has marked as hidden

Hidden achievements don't reveal their unlock conditions in Steam. Enable this if you prefer to discover and unlock these naturally yourself, while still automating the rest.

## Schedule Next Task
Automatically start another feature when Achievement Unlocker finishes

Select which feature should start when unlocking is complete — either Card Farming or Automatic Idler. Enable the checkbox to activate this, then choose your next task from the dropdown. See [Task Scheduling](/docs/features/task-scheduling) for more details.

## Time Range
Restrict achievement unlocking to a specific time window

When enabled, SGI will only unlock achievements while inside your scheduled time range. Outside of those hours, the Achievement Unlocker continues running but pauses unlocking until the window opens again.

For example, setting a range of `09:00` to `17:00` means SGI will only unlock achievements during business hours and wait overnight without stopping.

## Unlock Interval
Set how long SGI waits between unlocking each achievement *(in minutes)*

You specify a minimum and maximum range. After unlocking an achievement, SGI picks a random wait time within that range before unlocking the next one. This randomization makes the unlock pattern look more natural.

For example, with a range of `60` to `180` minutes, SGI will wait between 1 and 3 hours between each unlock.

- **Minimum value:** 1 minute
- **Maximum value:** 2880 minutes (48 hours)

Per-achievement delays set in the [Custom Order](/docs/features/achievement-unlocker#custom-order--unlock-delay) page override this global interval for individual achievements.
