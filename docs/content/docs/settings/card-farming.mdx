# Card Farming Settings

## Farm Games In List
Only farm cards for games you've manually added to your Card Farming list

SGI will ignore all other games in your library. Use this mode when you want precise control over which games get farmed.

<Callout type="info">
  **Farm Games In List** and **Farm All Games** are mutually exclusive — enabling one will disable the other.
</Callout>

## Farm All Games
Automatically farm cards for every game in your library that still has drops remaining

SGI will go through all eligible games without you needing to add them individually. You can check which games have drops remaining on your [Steam Badges page](https://steamcommunity.com/my/badges/).

<Callout type="info">
  **Farm Games In List** and **Farm All Games** are mutually exclusive — enabling one will disable the other.
</Callout>

## Skip Unplayed Games
Skip games that have never been played *(0 playtime)* during card farming

<Callout type="info">
  **Skip Unplayed Games** and **Unplayed Games Only** are mutually exclusive — enabling one will disable the other.
</Callout>

## Unplayed Games Only
Only farm cards for games that have never been played *(0 playtime)*

<Callout type="info">
  **Skip Unplayed Games** and **Unplayed Games Only** are mutually exclusive — enabling one will disable the other.
</Callout>

## Highest Drops First
Prioritize games with the most card drops remaining, farming them before games with fewer drops

<Callout type="info">
  **Highest Drops First** and **Lowest Drops First** are mutually exclusive — enabling one will disable the other.
</Callout>

## Lowest Drops First
Prioritize games with the fewest card drops remaining, farming those first

<Callout type="info">
  **Highest Drops First** and **Lowest Drops First** are mutually exclusive — enabling one will disable the other.
</Callout>

## Schedule Next Task
Automatically start another feature when Card Farming finishes

Select which feature should start when farming is complete — either Achievement Unlocker or Automatic Idler. Enable the checkbox to activate this, then choose your next task from the dropdown. See [Task Scheduling](/docs/features/task-scheduling) for more details.
