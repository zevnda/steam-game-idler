### Improved 🚀
- **Auto-Updates on Launch**: The `installer` version of SGI will now automatically download and install any available update when launched. If an update is detected while SGI is already running, the update button will still appear in the titlebar so you can apply it manually to prevent interruption
  - As SGI grows and more users onboard, supporting users across a wide range of versions has become one of the more time-consuming challenges of solo development. Users on older releases *(sometimes as far back as v2)* may be missing months of bug fixes and improvements, making support tickets harder to diagnose and resolve. Keeping everyone on the latest version helps lighten this load significantly
  - If you are adamant about staying on an older version of SGI, you can switch to using a `portable` build of the version you wish to remain on. You can download the portable build for each version on the [GitHub releases page](https://github.com/zevnda/steam-game-idler/releases)
