### Improved 🚀
- **Auto-Updates on Launch**: The `installer` build of SGI will now automatically download and install any available update when launched. If an update is detected while SGI is already running, the update button will still appear in the titlebar so you can apply it manually to prevent interruption
  - Supporting users across a wide range of versions is one of the bigger challenges of solo development. Users on older releases *(sometimes as far back as v2)* may be missing months of bug fixes, making tickets harder to resolve. Keeping everyone up to date helps lighten this load significantly
  - Older versions may no longer receive support. If you still wish to remain on an older version, you can switch to a `portable` build, available for each version on the [GitHub releases page](https://github.com/zevnda/steam-game-idler/releases)
