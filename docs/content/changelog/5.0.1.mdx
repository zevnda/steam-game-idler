### Improved 🚀
- **Achievement Unlocker**: 
  - Redesigned the "**Custom Order & Unlock Delay**" UI to be more intuitive and user-friendly. The UI is now opened in its own page, similar to the "**Achievement Manager**" page
  - Added a new "**Initial Delay**" option that allows users to set a custom delay before SGI starts unlocking achievements for a game, providing more flexibility in how achievements are unlocked
  - Now skips the 2 minute delay for games that have no achievements to unlock, as well as games with a custom "**Initial Delay**" configured
- **Inventory Manager**:
  - Improved the sorting and filtering for better clarity and usability
  - The "**List All**" button now lists only those items that match the current active filters, allowing for more targeted listing of items
  - Locked items are now hidden from the main view and can be accessed and unlocked via a new "**Is Locked**" filter
- **Achievement Manager**: Redesigned the UI to match the design of the "**Custom Order & Unlock Delay**" page
