### Improved 🚀
- Minor UI improvements to "**Trading Card Manager**". When a card is locked, the checkbox and price input are now disabled, and the "**List All (X)"** button will now count only non-locked cards
- Added stepper buttons to the "**Settings > Achievement Unlocker > Unlock Interval**" slider to allow for easier adjustments for both minimum and maximum values

### Fixed 🐛
- Fixed an issue in "**Achievements Manager**" which caused a small portion of the bottom of the list to be cut off
- Fixed an issue where the search bar was incorrectly disabled when opening "**Achievements Manager**" via the "**Card Farming**", "**Achievement Unlocker**" or "**Automatic Idler**" pages
