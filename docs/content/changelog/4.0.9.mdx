### New 🎉
- Added a checkbox to each achievement in the "**Achievements Manager**" list to allow for selecting multiple achievements at once. This enables the new "**Apply Changes**" button, which allows users to unlock or lock all selected achievements in one action
- Added 3 new settings to the "**Card Farming**" settings page:
  - **Unplayed Games Only**: Only farm cards for games that have not been played yet (0 playtime)
  - **Highest Drops First**: Prioritize farming games with the most card drops remaining first
  - **Lowest Drops First**: Prioritize farming games with the fewest card drops remaining first

### Improved 🚀
- Removed the max cap of 99,999 for the value inputs in the "**Statistics**" tab of "**Achievement Manager**"
- Improved the layout and styling of the achievement list in the "**Custom Order**" modal for "**Achievement Unlocker**", added a **select/deselect all** checkbox to the header, and already-unlocked achievements are now visually distinct with their controls disabled
