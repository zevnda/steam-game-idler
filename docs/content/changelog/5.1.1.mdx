### Improved 🚀
- The **All Games** tab in **Custom Lists** now has a sort dropdown. Games are sorted by a-Z by default
- Each game in the **Automatic Idler** list now has a checkbox to toggle it on or off without removing it from the list. Unchecked games are skipped when idling starts, both on launch and when using the **Start Manually** button
