### Improved 🚀
- **Custom Lists Redesigned**: The "**Card Farming**", "**Achievement Unlocker**", "**Automatic Idler**", and "**Favourite Games**" pages have been redesigned to improve the user experience. What was changed:
  - Removed the "**Edit List**" button and modal — everything is now managed directly on the page
  - Added a **tabbed layout** with three tabs: "**All Games**" (your full library), "**Current List**" (your curated list), and "**Blacklist**" (Card Farming only)
  - The "**All Games**" tab shows your entire game library as a searchable list with add/remove and blacklist controls per row
  - The "**Current List**" tab remains the same and preserves the existing drag-and-drop card grid for reordering
  - The "**Blacklist**" tab shows all blacklisted games and allows removing them individually or all at once via the **Clear** button
  - Blacklisted games can no longer be added to the list, and games already in the list can no longer be blacklisted
  - The list now display a **coloured indicator** — green when the game is in your list, red when blacklisted
