### New 🎉
- **Automatically Update Games List**: Added a new "**Automatically Update Games List**" setting to the "**General settings**" tab for "**PRO Gamer Tier**" users, which automatically updates the games list when SGI is launched, and then every 15 minutes while SGI is running, ensuring that any new games added to your Steam library are also added to SGI without needing to manually refresh
- **Easily List Duplicate Inventory Items**: Added a new "**List Duplicates**" button to "**Inventory Manager**" for "**PRO Gamer Tier**" users, allowing them to easily sell duplicate inventory items directly through SGI, providing a convenient way to automate the process of managing and monetizing excess items without leaving the app

### Improved 🚀
- **Tiered PRO Subscriptions**: 
  - The previous "**Legendary**" tier has been removed and now has two distinct tiers — "**Casual**" and "**Gamer**" — each unlocking a different set of benefits
    - **Casual**: Ad-free experience, exclusive themes, Discord role
    - **Gamer**: Everything in Casual, plus automated Steam credentials, automatic games list updates, automated free game redemption, and easily sell duplicate inventory items
- **Existing Subscribers Keep Their Benefits**: All users whose subscriptions were purchased and active before **April 10, 2026** retain access to all **PRO** features for as long as their subscription remains active — no action is required on their part
